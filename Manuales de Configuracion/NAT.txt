***NAT ESTATICO***

*Crear asignacion (que ip LAN va a internet)*
R2(config)# ip nat inside source static 192.168.10.254 209.165.201.5

*Configurar interfaz de manera interna*
R2(config)# interface serial 0/1/0
R2(config-if)# ip address 192.168.1.2 255.255.255.252
R2(config-if)# ip nat inside

*Configurar interfaz de manera externa*
R2(config)# interface serial 0/1/1
R2(config-if)# ip address 209.165.200.1 255.255.255.252
R2(config-if)# ip nat outside

***NAT DINAMICO***

*Definir pool de direcciones NAT*
R2(config)# ip nat pool NAT-POOL1 209.165.200.226 209.165.200.240 netmask 255.255.255.224

*Crear ACL para las ip locales internas*
R2(config)# access-list 1 permit 192.168.0.0 0.0.255.255 (utiliza wildmask)

*Enlazar ACL al grupo de direcciones NAT*
R2(config-if)# ip nat inside source list 1 pool NAT-POOL1

*Definir que interfaz esta dentro (dentro de la red)*
R2(config)# interface serial 0/1/0
R2(config-if)# ip nat inside

*Definir que interfaz esta fuera (fuera de la red)*
R2(config)# interface serial 0/1/1
R2(config-if)# ip nat outside

***PAT***

*Configuracion de PAT con una unica direccion*
R2(config)# ip nat inside source list 1 interface serial 0/1/1 overload
R2(config)# access-list 1 permit 192.168.0.0 0.0.255.255
R2(config)# interface serial0/1/0
R2(config-if)# ip nat inside
R2(config-if)# exit
R2(config)# interface Serial0/1/1
R2(config-if)# ip nat outside

*PAT con rango de direcciones*
R2(config)# ip nat pool NAT-POOL2 209.165.200.226 209.165.200.240 netmask 255.255.255.224
R2(config)# access-list 1 permit 192.168.0.0 0.0.255.255
R2(config)# ip nat inside source list 1 pool NAT-POOL2 overload
R2(config)#
R2(config)# interface serial0/1/0
R2(config-if)# ip nat inside
R2(config-if)# exit
R2(config)# interface serial0/1/1
R2(config-if)# ip nat outside
R2(config-if)# end
R2#

*Utilizar interfaz conectada para NAT de dispositivos internos*
R2(config)# ip nat inside source list 2 interface s0/1/1 overload
